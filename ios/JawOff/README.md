# JawOff iOS

SwiftUI版のiPhone MVPです。

## 目的

- 選んだ頻度で「歯、触れていませんか？」と通知する
- 通知をきっかけにアプリを開く
- 今の歯の接触状態をワンタップでチェックして保存する
- Jaw Awareness Scoreで変化を見返す
- 朝ログと直近の推移を端末内で見返す

## 起動方法

1. `ios/JawOff/JawOff.xcodeproj` をXcodeで開く
2. `JawOff` schemeを選ぶ
3. 実機またはiPhone Simulatorを選ぶ
4. Runする

実機で通知を試す場合は、XcodeのSigning & Capabilitiesで自分のTeamを選んでください。

## 通知仕様

- 初回は設定画面から通知許可を依頼
- 許可後、選択した頻度でローカル通知を登録
- 通知タイトル: `歯、触れていませんか？`
- 通知本文: `唇は閉じる。歯は離す。舌は上顎。`
- 頻度: `30分ごと`、`45分ごと`、`1時間ごと`、`90分ごと`、`2時間ごと`、`ランダム通知 25〜55分`

ランダム通知では、通知のたびに25〜55分の間で次の通知時間を決めます。MVPではアプリを開いている間の動作を前提にしています。

## Jaw Awareness Score

Jaw Awareness Scoreは、その日のクイックチェックで「離れていた」と答えた割合を0〜100点で表す指標です。

- `離れていた` = 100点
- `触れていた` = 0点
- その日の平均点がスコア

例: 10回中7回「離れていた」なら70点です。TCH改善の目的は、完全に制御することではなく、気づく回数を増やして日中の歯の接触時間を減らすことです。

## 保存

データは端末内の `UserDefaults` に保存します。

- `checkLogs`
- `morningLogs`
- `settings`
- `reminderLogs`

## 注意

このアプリはセルフケア支援を目的としたもので、診断・治療を行うものではありません。痛み、顎関節症状、歯の違和感が強い場合は歯科医師に相談してください。
