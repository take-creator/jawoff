# JawOff（歯を離す）

食いしばり・歯列接触癖（TCH）改善のためのセルフケア支援アプリです。ログイン不要のMVPとして、スマホ表示を優先して作っています。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 主な機能

- ホーム: 今日のJaw Awareness Score、チェック回数、触れていた/離れていた回数、前日比、7日平均を表示
- クイックチェック: 「今、上下の歯は触れていましたか？」にワンタップで回答
- 詳細チェック: 歯の接触、顎の力、舌の位置、肩首の力、ストレスを記録
- 朝ログ: 顎のだるさ、エラの張り感、歯の疲れ、頭痛、肩こり、睡眠の質、メモを保存
- グラフ: 直近7日/30日のJaw Awareness Score、チェック回数、触れていた率を表示
- 設定: 30分、45分、1時間、90分、2時間、ランダム通知 25〜55分のリマインダー頻度と通知許可
- 学習: TCH改善に関する短いカードと月1写真記録ガイド

## iPhoneアプリ版

SwiftUI版のiPhone MVPを `ios/JawOff` に追加しています。

- 30分、45分、1時間、90分、2時間、ランダム通知 25〜55分のローカル通知
- 通知文言: `歯、触れていませんか？`
- 通知本文: `唇は閉じる。歯は離す。舌は上顎。`
- 通知タップでチェック画面へ移動
- チェックログ、朝ログ、設定を端末内保存

## Jaw Awareness Score

Jaw Awareness Scoreは、その日のクイックチェックで「離れていた」と答えた割合を0〜100点で表す指標です。

- `離れていた` = 100点
- `触れていた` = 0点
- その日の平均点がスコア

例: 10回中7回「離れていた」なら70点です。TCH改善の目的は、完全に制御することではなく、気づく回数を増やして日中の歯の接触時間を減らすことです。

## リマインダー頻度

設定画面で `30分ごと`、`45分ごと`、`1時間ごと`、`90分ごと`、`2時間ごと`、`ランダム通知 25〜55分` を選べます。初期値は30分ごとです。

ランダム通知では、通知のたびに25〜55分の間で次の通知時間を決めます。MVPではアプリを開いている間の動作を前提にしています。

起動方法:

1. `ios/JawOff/JawOff.xcodeproj` をXcodeで開く
2. `JawOff` schemeを選ぶ
3. 実機またはiPhone SimulatorでRunする
4. 実機通知を使う場合は、XcodeのSigning & Capabilitiesで自分のTeamを選ぶ

## データ保存

データはブラウザの localStorage に保存します。

- `checkLogs`
- `morningLogs`
- `settings`
- `reminderLogs`

`reminderLogs` はホーム画面の「今日のリマインダー達成数」を表示するためのMVP用ログです。

## 注意

このアプリはセルフケア支援を目的としたもので、診断・治療を行うものではありません。痛み、顎関節症状、歯の違和感が強い場合は歯科医師に相談してください。
